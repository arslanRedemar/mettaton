## 1. 활동 포인트

서버 내 활동에 따라 포인트가 자동으로 적립됩니다. 7가지 활동 유형별로 독립적인 쿨다운이 적용됩니다.

### 포인트 적립 방식 (자동)

| 활동 | 포인트 | 쿨다운 | 적립 조건 |
|---|---|---|---|
| 포럼 글쓰기 | 300P | 5분 | 포럼 채널 스레드에 메시지 작성 |
| 질문 답변 | 300P | 5분 | `/질문답변` 명령어 실행 |
| 수행모임 참여 | 250P | 5분 | 수행모임 공지에 ✅ 리액션 |
| 개인수행 | 150P | 24시간 (하루 1회) | `/개인수행 체크` 또는 ✅ 리액션 |
| 일반활동 | 50P | 5분 | 메시지 전송, 리액션 추가 |
| 퀴즈 참가 | 150P | 하루 1회 | 일일 퀴즈 답변 제출 |
| 퀴즈 정답 | 200P | 하루 1회 | 일일 퀴즈 정답 시 (참가 포인트 대체) |

> 포인트는 별도 알림 없이 자동으로 적립됩니다. 봇의 메시지와 리액션은 적립 대상에서 제외됩니다.

### 내 포인트 확인

```
/포인트
```

현재 보유한 총 포인트를 확인합니다. 본인에게만 표시됩니다.

### 포인트 내역 조회

```
/포인트내역
```

활동 유형별 포인트 적립 내역을 파이차트로 확인합니다.

**옵션:**
| 옵션 | 설명 | 필수 |
|---|---|---|
| 시작일 | 조회 시작일 (YYYY-MM-DD) | 아니오 |
| 종료일 | 조회 종료일 (YYYY-MM-DD) | 아니오 |

- 날짜를 지정하지 않으면 전체 누적 내역이 표시됩니다.
- 날짜를 지정하면 해당 기간 내 적립 내역만 표시됩니다.

파이차트 색상 구분:
- 포럼 글쓰기: 초록
- 질문 답변: 파랑
- 수행모임 참여: 주황
- 개인수행: 보라
- 일반활동: 회색
- 퀴즈 참가: 분홍
- 퀴즈 정답: 빨강

## 2. 일일 퀴즈

매일 정해진 시간에 문제은행 채널에 5지선다 객관식 문제가 자동으로 출제됩니다. 해설이 공개되기 전까지 답변을 제출하세요.

### 답변 제출

```
/답변 [번호]
```

| 옵션 | 설명 | 필수 |
|---|---|---|
| 번호 | 정답 번호 선택 (1~5) | 예 |

- 해설 공개 전까지만 제출 가능합니다.
- 해설 공개 전이라면 **답변 수정이 가능**합니다.
- 한 문제당 사용자별 1건만 제출됩니다.

**포인트:**
- 참가 포인트: 150P (답변 제출 시)
- 정답 포인트: 200P (참가 포인트를 대체하며, 중복 지급되지 않습니다)
- 포인트는 해설 공개 시점에 일괄 지급됩니다.

### 내 답변 확인

```
/내답변
```

오늘 출제된 문제에 대한 내 답변과 제출 시각을 확인합니다. 본인에게만 표시됩니다.

### 일일 퀴즈 흐름

1. 정해진 시간에 문제은행 채널에 문제가 자동 출제됩니다.
2. `/답변 [1~5]`으로 답변을 제출합니다.
3. 정해진 시간에 해설이 자동 공개됩니다.
4. 해설 공개 시점에 포인트가 정산됩니다.

## 3. 개인 수행

개인 수행 계획(명상, 절, 독서 등)을 등록하고, 매일 체크인하며, 달력 스탬프 또는 히트맵으로 진행 현황을 확인할 수 있습니다.

### 수행 계획 등록

```
/개인수행 등록
```

| 옵션 | 설명 | 필수 |
|---|---|---|
| 수행내용 | 수행할 내용 | 예 |
| 매일목표 | 하루 목표량 | 예 |
| 단위 | 단위 (분, 회, 권 등) | 예 |
| 시작일 | 시작 날짜 (YYYY-MM-DD) | 예 |
| 종료일 | 종료 날짜 (YYYY-MM-DD) | 예 |

- 계획 기간은 최대 365일까지 등록 가능합니다.
- 시작일은 오늘 이후여야 합니다.
- 등록 시 수행계획방 채널에 Embed 메시지가 게시됩니다.
- 여러 개의 수행 계획을 동시에 운영할 수 있습니다.

### 일일 체크인

**방법 1: 슬래시 명령어 (모든 채널에서 사용 가능)**
```
/개인수행 체크
```
자동완성 목록에서 수행 계획을 선택하여 체크인합니다.

**방법 2: 리액션**
수행계획방 채널에 있는 내 수행 계획 Embed 메시지에 ✅ 리액션을 추가합니다.

- 하루에 계획당 1회만 체크 가능합니다.
- ✅ 리액션을 제거하면 당일 기록이 취소됩니다.
- 계획 기간 내에서만 체크인이 유효합니다.
- 체크인 시 150P가 적립됩니다 (하루 1회 제한).

### 수행 현황 확인

```
/개인수행 확인
```

수행 현황을 이미지로 확인합니다:
- **7개월 미만:** 달력 스탬프 (월별 그리드, 완료일은 초록 스탬프)
- **7개월 이상:** GitHub 스타일 히트맵
- 달성률 프로그레스 바가 함께 표시됩니다.
- 오늘 날짜는 파란 테두리로 강조됩니다.

### 수행 계획 수정

```
/개인수행 수정
```

등록된 수행 계획을 수정합니다. 시작일은 변경할 수 없습니다.

### 수행 계획 삭제

```
/개인수행 삭제
```

등록된 수행 계획을 삭제합니다. 수행계획방의 Embed 메시지도 함께 삭제됩니다.

> 모든 개인수행 명령어는 본인의 계획에만 적용됩니다.

## 4. 질문 관리

질문을 등록하고, 목록을 조회하고, 답변하거나 삭제할 수 있습니다. 등록된 질문은 질문 채널에 Embed 메시지로 게시됩니다.

### 질문 등록

```
/질문등록
```

| 옵션 | 설명 | 필수 |
|---|---|---|
| 내용 | 질문 내용 | 예 |

- 질문 채널에 주황색 Embed가 게시됩니다 (미답변 상태).
- 다른 사용자가 ✅ 리액션으로 관심을 표시할 수 있습니다.

### 질문 목록 조회

```
/질문목록
```

등록된 질문 목록을 확인합니다. 작성자, 답변 상태, 관심 수가 표시됩니다. 본인에게만 표시됩니다.

### 질문에 답변하기

```
/질문답변
```

| 옵션 | 설명 | 필수 |
|---|---|---|
| id | 질문 ID | 예 |
| 내용 | 답변 내용 | 예 |

- Embed가 녹색으로 변경되며 답변 내용이 표시됩니다.
- 답변 시 300P가 적립됩니다.

### 질문 삭제

```
/질문삭제
```

| 옵션 | 설명 | 필수 |
|---|---|---|
| id | 질문 ID | 예 |

질문과 채널의 Embed 메시지가 함께 삭제됩니다.

## 5. 일정

등록된 일정을 조회하고, 리액션으로 참석 여부를 표시할 수 있습니다.

### 일정 목록 조회

```
/일정목록
```

등록된 일정 목록을 확인합니다. 날짜, 시간, 장소, 주최자, 참석자 수가 표시됩니다. 본인에게만 표시됩니다.

### 리액션으로 참석 표시

일정 채널에 게시된 일정 Embed 메시지에서:
- ✅ 리액션: 참석 등록
- ❌ 리액션: 참석 취소
- ✅ 리액션 제거: 참석 취소

Embed의 참석자 수가 자동으로 갱신됩니다.

## 6. 수행 모임

매일 정해진 시간에 지정된 채널에 수행 모임 알림이 자동으로 게시됩니다.

### 리액션으로 참석 표시

수행 모임 알림 메시지에서:
- ✅ 리액션: 참석 표시 (250P 적립)
- ❌ 리액션: 불참 표시
- ✅ 리액션 제거: 참석 취소

알림 게시 후 6시간 동안 참석을 받으며, 이후 자동으로 완료 처리됩니다.

## 7. 달 위상 달력

현재 월의 달 위상 달력을 이미지로 확인합니다.

### 명령어

```
!달위상
```

- 슬래시 명령어가 아닌 **텍스트 명령어**입니다. 채팅에 직접 입력하세요.
- 서울 기준 달 위상 달력 이미지가 Embed로 표시됩니다.
- 월별 캐싱으로 동일 월 내 중복 요청이 방지됩니다.
